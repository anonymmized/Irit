# Changelog

All notable changes to `Irit` should be documented in this file.

## [0.7.0] - 2026-04-06

### Added

- Rebuilt `fastserver.sh` as a working Irit core.
- Added `doctor` mode for non-destructive diagnostics.
- Added richer client bundle export:
  - `vless-uri.txt`
  - `client-template.json`
  - `sing-box-client.json`
  - `mihomo-profile.yaml`
  - `manifest.json`
  - `import-hints.txt`
  - QR assets
- Added local artifact download flow after setup or access recovery.
- Added clipboard copy support for the generated VLESS URI.
- Added stronger terminal UX with boxed summaries and cleaner status output.
- Added GitHub growth-pack files, landing page, CI, and repo community setup.
